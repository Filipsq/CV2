Projekt Profil Zawodowy
Opis Projektu
Projekt polega na stworzeniu strony internetowej prezentującej profil zawodowy osoby, używając technologii JavaScript oraz frameworków takich jak React, React Router, i Axios. 
Strona zawiera sekcje takie jak wykształcenie, języki, doświadczenie, umiejętności oraz ukończone projekty. 
Główne funkcjonalności obejmują interaktywne elementy, nawigację między podstronami oraz dynamiczne pobieranie danych.

Problemy i ich Rozwiązania
Problem: Problemy z renderowaniem komponentów w React po wprowadzeniu zmian.
Rozwiązanie: Użycie stanu komponentów i hooków, takich jak useState i useEffect, do zarządzania cyklem życia komponentu.

Problem: Błędy przy nawigacji między stronami.
Rozwiązanie: Zastosowanie React Router do zarządzania ścieżkami i dynamicznego renderowania komponentów na podstawie URL.

Problem: Błędy przy wykonywaniu zapytań HTTP.
Rozwiązanie: Użycie Axios do obsługi zapytań HTTP, wraz z blokami try-catch do obsługi błędów.

Opis Plików
HTML (index.html)
Plik HTML zawiera strukturę strony internetowej z podziałem na sekcje takie jak:

Nagłówek z nazwą i zdjęciem profilowym
Sekcje edukacji, języków, doświadczenia, umiejętności oraz ukończonych projektów
Przycisk do przełączania trybu ciemnego
Skrypt JavaScript osadzony na końcu dokumentu

CSS (style.css)
Plik CSS definiuje stylizację strony, w tym:

Stylizację nagłówka, sekcji oraz elementów list
Stylizację przycisku przełączania trybu ciemnego
Media queries dla responsywności

JavaScript (script.js)
Plik JavaScript zawiera logikę interaktywną strony, w tym:

Stylizację nagłówka oraz profilowego zdjęcia
Przełączanie trybu ciemnego
Pasek postępu przewijania strony

Podsumowanie
Projekt wykorzystuje najnowsze technologie webowe, aby stworzyć interaktywną stronę profilową.
